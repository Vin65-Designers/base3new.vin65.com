<v65:wine:type></v65:wine:type>

The v65:wine:type tag displays the wine Type value.

Available on:

  • Product List
  • Product QuickView
Type
Red
The v65:wine:type tag outputs the following HTML code:
<div id="v65-wine-type">
	<div class="v65-wine-specLabel">Type</div>
	<div class="v65-wine-specValue">Red</div>
</div>
