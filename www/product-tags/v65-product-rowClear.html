The v65:product:rowClear displays creates a break after a set number of products.

Optional Attribute(s):

clearAfter:   <v65:product:rowClear clearAfter="2"></v65product:rowClear>

Available on:

  • Product List
 
The v65:product:rowClear tag outputs the following HTML code:
<div class="v65-product-rowClear"> </div>
